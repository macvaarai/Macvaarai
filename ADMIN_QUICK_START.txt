HOW TO START
═══════════════════════════════════════════════════════════════════════════════

STEP 1: Initialize Database
  Location: c:\bhai health\macvaarai-backend\database_schema.sql
  Already created with all tables and default admin user

STEP 2: Update App.jsx
  Add these imports at top:
    import AdminLogin from "./Components/AdminLogin.jsx";
    import AdminDashboard from "./Components/AdminDashboard.jsx";

  Add wrapper function:
    const AdminWrapper = ({ children }) => {
      const adminToken = localStorage.getItem("adminToken");
      if (!adminToken) return <Navigate to="/admin/login" />;
      return children;
    };

  Add these routes (before closing </Routes>):
    <Route path="/admin/login" element={<AdminLogin />} />
    <Route path="/admin/dashboard" element={
      <AdminWrapper>
        <AdminDashboard />
      </AdminWrapper>
    } />

STEP 3: Start Backend
  Terminal 1:
    cd "c:\bhai health\macvaarai-backend"
    python -m uvicorn main:app --reload

STEP 4: Start Frontend
  Terminal 2:
    cd "c:\bhai health\macvaarai-frontend\macvaarai-frontend"
    npm run dev

STEP 5: Access Admin Panel
  Browser:
    http://localhost:5173/admin/login

  Credentials:

    Password: anbu@1001
