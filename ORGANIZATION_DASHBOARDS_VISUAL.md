## 🖼️ HOW TO ADD LOGOS

### **Step 1: Create Logos Folder**
```

├─ vijay-logo-white.png

├─ modi-logo.png

├─ cbn-logo.png
└─ cbn-logo-white.png
```

### **Step 2: Update Database**

```sql
UPDATE organizations SET logo_url = '/org-logos/vijay-logo.png' 
WHERE name = 'Vijay Care';

UPDATE organizations SET logo_url = '/org-logos/bjp-logo.png' 
WHERE name = 'BJP Care';

UPDATE organizations SET logo_url = '/org-logos/modi-logo.png' 
WHERE name = 'Modi Care';

UPDATE organizations SET logo_url = '/org-logos/cbn-logo.png' 
WHERE name = 'CBN Care';
```

### **Step 3: Display Logo in Dashboard**

```javascript
// React Component

const OrganizationDashboard = ({ organization }) => {
  const getThemeColor = (orgName) => {
    switch(orgName) {
      case 'Vijay Care':
        return {
          primary: '#FFD700',
          secondary: '#FFA500',
          bg: '#FFFACD'
        };
      case 'BJP Care':
        return {
          primary: '#FF9933',
          secondary: '#FFFFFF',
          bg: '#FFF3E0'
        };
      case 'Modi Care':
        return {
          primary: '#003399',
          secondary: '#FF9933',
          bg: '#E8F4F8'
        };
      case 'CBN Care':
        return {
          primary: '#0066CC',
          secondary: '#FFFFFF',
          bg: '#E6F2FF'
        };
      default:
        return {
          primary: '#003D82',
          secondary: '#0066CC',
          bg: '#F5F5F5'
        };
    }
  };

  const theme = getThemeColor(organization.name);

  return (
    <div style={{ backgroundColor: theme.bg }}>
      {/* Header */}
      <div style={{ backgroundColor: theme.primary }}>
        <img 
          src={organization.logo_url} 
          alt={organization.name}
          style={{ height: '60px', width: 'auto' }}
        />
        <h1 style={{ color: theme.secondary }}>
          {organization.name}
        </h1>
        <p>{organization.tagline}</p>
      </div>

      {/* Statistics */}
      <div style={{ display: 'grid', gridTemplateColumns: '1fr 1fr 1fr 1fr' }}>
        <div style={{ 
          backgroundColor: theme.secondary,
          borderLeft: `5px solid ${theme.primary}`
        }}>
          <h3>Total Beds</h3>
          <p>{organization.totalBeds}</p>
        </div>
        {/* More stat cards */}
      </div>

      {/* Hospitals List */}
      <div>
        <h2 style={{ color: theme.primary }}>
          Hospitals Under {organization.name}
        </h2>
        {organization.hospitals.map((hospital) => (
          <div key={hospital.id} style={{
            borderLeft: `4px solid ${theme.primary}`,
            padding: '10px',
            marginBottom: '10px'
          }}>
            <h3>{hospital.name}</h3>
            <p>Beds: {hospital.beds} | Patients: {hospital.patients}</p>
          </div>
        ))}
      </div>
    </div>
  );
};
```

## 🎨 LOGO GUIDELINES

### **Logo Requirements:**

```
Format: PNG, JPG, SVG
Size: Recommended 300x300px minimum
Colors: Match organization branding
Style: Clean, professional
Variations:
├─ Full color logo
├─ White logo (for dark backgrounds)
├─ Black logo (for light backgrounds)
└─ Icon-only version (for small spaces)
```

### **Logo Placement:**

```
Dashboard Header:

└─ Logout button (right)

Favicon:
└─ Icon-only version
```
